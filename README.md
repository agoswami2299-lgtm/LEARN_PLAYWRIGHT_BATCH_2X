# LEARN_PLAYWRIGHT_BATCH_2X

Welcome to the **LEARN_PLAYWRIGHT_BATCH_2X** repository!  
This project contains hands-on JavaScript and Playwright learning materials organized chapter-wise.

---

## Repository Structure

```
LEARN_PLAYWRIGHT_BATCH_2X/
├── Chapter_01_Basics/                  # JavaScript Basics & Setup
│   ├── 01_Basic.js
│   ├── 02_Basic.js
│   ├── 03_JS_Verify_Setup.js
│   └── 04_Hot_Code.js
│
├── Chapter_02_Javascript_Concept/      # Core JavaScript Concepts
│   └── 05_JS_Basic
│
└── Chapter_03_Identifier_Literlas/     # Identifiers, Literals & Comments
    ├── 06_Identifier_Rules.js
    ├── 07_Identifier_Part2.js
    └── Comments.js
```

---

## Chapters Overview

### Chapter 01 — Basics
Introduction to JavaScript fundamentals, environment setup verification, and basic scripting examples.

### Chapter 02 — JavaScript Concepts
Core programming concepts to build a solid foundation before diving into Playwright automation.

### Chapter 03 — Identifiers & Literals
Learn the rules for naming variables, valid identifier patterns, literals, and how to write meaningful comments in JavaScript.

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/agoswami2299-lgtm/LEARN_PLAYWRIGHT_BATCH_2X.git
   cd LEARN_PLAYWRIGHT_BATCH_2X
   ```

2. **Run a JavaScript file (Node.js)**
   ```bash
   node Chapter_01_Basics/01_Basic.js
   ```

## Prerequisites

- [Node.js](https://nodejs.org/) installed on your machine
- Basic understanding of programming concepts
